## Summary (132 characters max)
Detect AI-generated content and bot activity on any webpage with real-time analysis, visual highlighting, and detailed statistics.

## Description

**Identify AI-Generated Content and Bot Activity Instantly**

PoC AI Content Detector helps you distinguish between human-written and AI-generated content across the web. With powerful detection algorithms and real-time analysis, you'll know what's real and what's not.

**Key Features:**

✓ **Real-Time AI Detection** - Automatically scan webpages for AI-generated content as you browse

✓ **Visual Highlighting** - Color-coded indicators show AI vs. human content at a glance

✓ **Multi-Platform Support** - Works on Twitter/X, Reddit, LinkedIn, and general web content

✓ **Detailed Statistics** - Track your total scans and see AI detection trends

✓ **Privacy-Focused** - All data stored locally on your device

✓ **Customizable** - Configure your own backend API server

**How It Works:**

1. Install the extension
2. Configure your API server URL in settings
3. Click "Scan This Page" or let it auto-scan content
4. View results with color-coded highlights and detailed analysis

**Perfect For:**

- Researchers studying AI-generated content
- Content moderators identifying automated posts
- Anyone curious about AI content on the web
- Developers testing AI detection systems

**Privacy & Security:**

This extension requires a backend API server for AI detection analysis. You can run the server locally or deploy it to your preferred hosting service. All scan data is stored locally on your device. We do not collect, transmit, or store any of your browsing data.

**Open Source:**

PoC AI Content Detector is fully open source. Review the code, contribute improvements, or deploy your own instance.

**Support:**

For setup instructions, documentation, and support, visit our GitHub repository or open an issue.

**Note:** This extension requires a compatible backend server to function. See the documentation for setup instructions.

## Privacy Practices

**Data Usage:**
- The extension stores scan statistics and settings locally
- Text content from webpages is sent to your configured API server for analysis
- No data is collected or transmitted to the extension developers

**Permissions Justification:**
- **activeTab**: Required to read webpage content for AI detection
- **storage**: Required to save settings and scan statistics
- **scripting**: Required to inject content detection scripts
- **host_permissions (<all_urls>)**: Required to work on any website
